我在用 hexo 的 butterfly 主题搭建博客的时候，发现用markdown的语法插入的图片无法显示出来。当时的我也懒，就直接上网找解决方法。一顿操作之后——

好家伙，一口气找到好几篇一模一样的文章……也是绝了哈。没一篇有用的（要是有用，你可能就不会看到这篇文章了）。

1.下载插件

  • 用终端进入你博客的根目录
  • 键入命令 npm install hexo-asset-image --save

2.修改插件

作者写插件的时候是2018年，那个时候可能也就是 hexo 3.1 版本.

现在我安装的 hexo 版本已经是 4.3.0 了 (2022年).

可能会有问题到现在。

不废话，下面是步骤：

  • 进入你博客的根目录，然后下面顺序找到index.js:
